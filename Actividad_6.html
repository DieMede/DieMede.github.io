Siglas

Open Systems Interconnection

El Modelo OSI (de las siglas en inglés: Open Systems Interconnection, o sea, “Interconexión de Sistemas Abiertos”).Es una herramienta conceptual, de organización de las telecomunicaciones. Universaliza la manera en que la información es compartida entre redes informáticas o sistemas computarizados, independientemente de su origen geográfico, empresarial u otras condiciones que podrían dificultar la comunicación de los datos. Fuente: https://concepto.de/modelo-osi/#ixzz8yjSrZn2b

Internet Assigned Numbers Authority

Una dirección IP es una cadena de números separados por puntos. Las direcciones IP se expresan como un conjunto de cuatro números, por ejemplo, 192.158.1.38. Cada número del conjunto puede variar de 0 a 255. Por lo tanto, el rango completo de direcciones IP va desde 0.0.0.0 hasta 255.255.255.255. Las direcciones IP no son aleatorias. La Autoridad de números asignados de Internet (Internet Assigned Numbers Authority, IANA), una división de Internet Corporation para números y nombres asignados (Internet Corporation for Assigned Names and Numbers, ICANN), genera y asigna matemáticamente las direcciones IP.

Transmission Control Protocol

TCP (Protocolo de Control de Transmisión, por sus siglas en inglés Transmission Control Protocol) es protocolo de red importante que permite que dos anfitriones (hosts) se conecten e intercambien flujos de datos. TCP garantiza la entrega de datos y paquetes (en-US) en el mismo orden en que se enviaron.

User Datagram Protocol

UDP son las siglas de Protocolo de Datagrama de Usuario (en inglés User Datagram Protocol) un protocolo sin conexión que, como TCP, funciona en redes IP. UDP significa "Protocolo de Datagrama de Usuario" y es un protocolo de comunicación utilizado en redes de computadoras. Su función es transmitir datos en forma de paquetes de manera rápida y eficiente.

Secure Shell

SSH es un acrónimo de Secure Shell. Es un término genérico que se refiere a las diferentes versiones de los protocolos SSH. Los protocolos SSH especifican estándares para operar los servicios de red de forma segura entre anfitriones para los que no existe una relación de confianza a través de redes no seguras. Las comunicaciones entre un cliente y un servidor que utilizan SSH están encriptadas, por lo que es un protocolo idóneo para su uso en redes no seguras.

File Transfer Protocol

El FTP es un Protocolo de transferencia de archivos (por sus siglas FTP o File Transfer Protocol en inglés) es el conjunto de normas que indican a los ordenadores cómo transferir archivos entre distintos sistemas en línea.

Simple Mail Transfer Protocol

El SMTP es un protocolo que se utiliza para enviar y recibir correos electrónicos. SMTP es la sigla que corresponde a la expresión de la lengua inglesa Simple Mail Transfer Protocol. En nuestro idioma, dicho concepto puede traducirse como Protocolo para la Transferencia Simple de Correo.

Post Office Protocol

En informática, el POP (Post Office Protocol) es un protocolo estándar de internet de la capa aplicación (en el modelo OSI). El protocolo POP es utilizado por programas de e-mail locales para recibir e-mails desde un servidor remoto a través de una conexión TCP/IP.

Palabra

El DHCP (Protocolo de configuración dinámica de host) es un protocolo de administración de red utilizado para asignar dinámicamente una dirección de Protocolo de Internet (IP) a cualquier dispositivo o nodo en una red para que puedan comunicarse mediante IP.

Domain Name System

Las siglas DNS significan Domain Name System, que se traduce como Sistema de Nombres de Dominio. Es un método de denominación utilizado para nombrar dispositivos que se conectan a una red a través del Protocolo de Internet (IP).

HyperText Transfer Protocol

HTTP (del inglés HyperText Transfer Protocol o Protocolo de Transferencia de Hiper Textos) es el protocolo de transmisión de información de la World Wide Web. Permite la circulación de información a través de Internet y se utiliza para publicar páginas web o HTML.

Palabra

El protocolo ARP utiliza una forma simple para sus mensajes, y poder envías las solicitudes a la vez que responderlas. Principalmente estaba pensado para ser utilizado con direcciones IPv4 y MAC, pero esto también se puede utilizar con otros protocolos de red.

Media Access Control

Dirección MAC es un acrónimo de Media Access Control (Control de Acceso a Medios) y se refiere a un código único de 12 dígitos asignado a cada dispositivo conectado a una red local. Esta dirección permite identificarlo y permitir el intercambio de datos entre los equipos.

Red de área local

LAN son las siglas de Local Area Network, Red de área local. Una LAN es una red que conecta los ordenadores en un área relativamente pequeña y predeterminada (como una habitación, un edificio, o un conjunto de edificios).
